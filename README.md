Это простой, но функциональный проект чата, написанный на C++. Он состоит из сервера и клиента (консольного и веб-интерфейса), которые общаются через HTTP-запросы. Проект отлично подходит для изучения основ сетевого программирования, создания клиент-серверных приложений и работы с популярной библиотекой `cpp-httplib`.

## Возможности

*   **Мгновенный обмен сообщениями** между несколькими клиентами.
*   **Консольный клиент** для удобного использования в терминале.
*   **Веб-интерфейс** чата, доступный через браузер.
*   **Очистка истории чата** командой `/clear` в консольном клиенте.
*   **Временные метки** для каждого сообщения.
*   **Простота запуска** — минимум зависимостей.

## Технологии

*   **Язык:** C++11
*   **Сеть:** [cpp-httplib](https://github.com/yhirose/cpp-httplib) — легковесная, однофайловая библиотека для работы с HTTP.
*   **Система сборки:** CMake

## Как это работает

Сервер хранит историю сообщений в памяти и предоставляет простой REST-like API.

### API Эндпоинты

*   `GET /` — возвращает HTML-страницу с веб-чатом.
*   `GET /api/messages` — возвращает JSON-список всех сообщений.
*   `POST /api/send` — принимает JSON с полями `username` и `message` для отправки нового сообщения.
*   `POST /api/clear` — очищает всю историю сообщений на сервере.
